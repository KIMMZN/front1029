자동화 장바구니 ui

기능구현:
1. 상품을 장바구니에 담기 기능
	- 상품 입력시 상품명, 수량, 배송비를 입력하고 주문하기 버튼을 누르면 
	 장바구니에 담기 가능
	- table의 row와 column을 추가해 화면에 표시함


2. 장바구니에 여러개 담기 가능
	- 주문하기 버튼을 클릭해 상품을 여려개 담기 가능
	

3. 상품의 가격과 배송비 자동계산 기능
	- 상품을 선택하면 가격과 배송비가 자동 계산됨

5. 전체 선택 기능
	- 전체선택 체크박스를 선택해 상품을 전체선택 가능

6. 상품 삭제 기능
	- 상품을 선택해 삭제 할수 있음
	- table의 row를 삭제해서 화면에서 삭제함

	

구현방법:

상품은 정해지지 않았고
문서가 로딩될때 장바구니에 담긴 상품이 없으면
"장바구니가 비어있습니다. 물건을 주문해주세요"라는
table row가 표시된다.

상품명을 적고 상품가격과 수량, 배송비를 입력하면 상품이 장바구니에 담긴다.
장바구니에서 상품 체크박스를 선택하면 상품가격과 배송비 총 주문금액이 자동 계산된다

전체선택 체크박스를 선택하면 상품이 전체 선택되고
전체선택된 상품의 가격도 자동 계산된다

전체선택해 상품을 전체 삭제 할수 있다.

장바구니에 담긴 상품이 하나도 존재하지 않는다면
"장바구니가 비어있습니다. 물건을 주문해주세요"라는
table row가 다시 표시된다

4.주요 사용기술
 
-순수한 자바스크립트의 기능을 사용하였음.

-상품등록시 addevetlistener를 사용해 click등 이벤트 발생시 
람다식 함수 표현식을 사용하고
dom구조를 이용해 html요소에서 값을 가져와
table의 row와 column을 생성하여 상품정보를 저장 하여 표시
상품등록시 상품의 체크박스에 for문을 사용하여 eventlistener에 등록
이때 이벤트 종류로 change를 사용하였음

-상품의 체크 박스를 선택하면 등록된 이벤트리스너의 함수가 호출되어
dom구조를 이용해 element 값을 가져오고
가격을 계산,

-계산시 parseFloat()메서드를 사용해 string을 실수로 변환하여 계산


-전체선택체크박스 클릭시 dom구조를 이용해 상품table의 모든 체크박스를
true/false로 바꿔서 선택or해제 되게함

-삭제시 for문을 사용하여 상품 table의 row를 조회, deleteRow메서드를 사용해
해당 상품의 row를 삭제.
